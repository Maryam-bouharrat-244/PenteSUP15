# pente_sup15

Script Python pour calculer les zones où la pente dépasse 15% à partir de MNT départementaux.

## Utilisation

1. Placez tous les fichiers MNT (un par département) dans le dossier `MNT`.
2. Exécutez le script `calcul_pente_sup15.py`.
3. Les résultats (rasters binaires) seront générés dans le dossier `PENTE_SUP15`.

## Dépendances
- rasterio
- numpy

Installez-les avec :
```
pip install rasterio numpy
```
